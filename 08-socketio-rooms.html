Rooms et Namespaces

Socket.io : Rooms

Rejoindre et quitter une room

io.on('connection', (socket) => {

  socket.on('room:join', (roomName: string) => {
    socket.join(roomName);

    // Confirmer à l'utilisateur qu'il a rejoint la room
    socket.emit('room:joined', roomName);

    // Notifier les autres membres de la room
    socket.to(roomName).emit('room:user-joined', {
      userId: socket.id,
      roomName
    });
  });

  socket.on('room:leave', (roomName: string) => {
    socket.leave(roomName);
    socket.to(roomName).emit('room:user-left', {
      userId: socket.id,
      roomName
    });
  });

});

Envoyer des messages dans une room

io.on('connection', (socket) => {

  socket.on('room:message', (data: {
    roomName: string;
    text: string;
    author: string;
  }) => {
    const message = {
      id: Date.now(),
      ...data,
      timestamp: new Date().toISOString()
    };

    // Envoyer à tous les membres de la room (y compris l'émetteur)
    io.to(data.roomName).emit('room:message', message);
  });

});

Lister les membres d'une room

async function getRoomMembers(roomName: string): Promise<string[]> {
  const sockets = await io.in(roomName).fetchSockets();
  return sockets.map(s => s.data.username ?? s.id);
}

io.on('connection', (socket) => {

  socket.on('room:get-members', async (roomName: string) => {
    const members = await getRoomMembers(roomName);
    socket.emit('room:members', { roomName, members });
  });

  socket.on('room:join', async (roomName: string) => {
    socket.join(roomName);
    const members = await getRoomMembers(roomName);
    // Mettre à jour la liste pour tous les membres
    io.to(roomName).emit('room:members', { roomName, members });
  });

});

Stocker des données sur le socket

La propriété socket.data permet de stocker des informations liées à un client :

io.on('connection', (socket) => {

  socket.on('user:set-name', (username: string) => {
    socket.data.username = username;
    socket.data.connectedAt = new Date();

    socket.emit('user:name-set', { username, id: socket.id });
    socket.broadcast.emit('user:connected', { username });
  });

  socket.on('disconnect', () => {
    if (socket.data.username) {
      io.emit('user:disconnected', { username: socket.data.username });
    }
  });

});

Gestion des rooms côté client

// Rejoindre une room
function joinRoom(roomName: string): void {
  socket.emit('room:join', roomName);
  currentRoom = roomName;
}

// Écouter les événements de la room
socket.on('room:message', (message: RoomMessage) => {
  if (message.roomName === currentRoom) {
    displayMessage(message);
  }
});

socket.on('room:members', (data: { roomName: string; members: string[] }) => {
  updateMembersList(data.roomName, data.members);
});

socket.on('room:user-joined', (data: { userId: string; roomName: string }) => {
  showNotification(`${data.userId} a rejoint ${data.roomName}`);
});

Historique des messages (en mémoire)

interface Message {
  id: number;
  roomName: string;
  author: string;
  text: string;
  timestamp: string;
}

const history = new Map<string, Message[]>();

function saveMessage(message: Message): void {
  const roomHistory = history.get(message.roomName) ?? [];
  roomHistory.push(message);
  // Garder les 50 derniers messages par room
  history.set(message.roomName, roomHistory.slice(-50));
}

// Envoyer l'historique lors d'une connexion à une room
socket.on('room:join', (roomName: string) => {
  socket.join(roomName);
  const roomHistory = history.get(roomName) ?? [];
  socket.emit('room:history', roomHistory);
});

Projet pratique : chat multi-rooms

Construire une application de chat complète avec :

  • Saisie d'un nom d'utilisateur à la connexion
  • Liste des rooms disponibles avec le nombre de participants
  • Possibilité de créer, rejoindre et quitter des rooms
  • Historique des 50 derniers messages par room
  • Indicateur "est en train d'écrire..."
  • Notifications de connexion / déconnexion

Exercice

  1. Implémenter le système de rooms dynamiques : les utilisateurs peuvent créer une room en tapant son nom
  2. Afficher la liste des rooms actives avec leur nombre de participants (mise à jour en temps réel)
  3. Ajouter la liste des participants dans le panneau latéral de chaque room
  4. Implémenter l'historique des messages : les 50 derniers messages s'affichent à l'arrivée dans une room
  5. Gérer proprement la déconnexion : retirer l'utilisateur de toutes ses rooms et notifier les membres
