WebSockets

Le protocole WebSocket

Le protocole WebSocket (RFC 6455, 2011) permet d'établir un canal de communication bidirectionnel full-duplex entre un client et un serveur sur une seule connexion TCP.

Contrairement à HTTP, la connexion reste ouverte : les deux parties peuvent s'envoyer des messages à tout moment.

RFC 6455 : The WebSocket Protocol

Le handshake HTTP → WebSocket

La connexion WebSocket démarre par une requête HTTP classique avec un header Upgrade :

GET /chat HTTP/1.1
Host: localhost:8080
Upgrade: websocket
Connection: Upgrade
Sec-WebSocket-Key: dGhlIHNhbXBsZSBub25jZQ==
Sec-WebSocket-Version: 13

Le serveur accepte la mise à niveau :

HTTP/1.1 101 Switching Protocols
Upgrade: websocket
Connection: Upgrade
Sec-WebSocket-Accept: s3pPLMBiTxaQ9kYGzzhZRbK+xOo=

Différences fondamentales avec HTTP

HTTP WebSocket
Requête/Réponse Canal permanent bidirectionnel
Connexion fermée après la réponse Connexion maintenue ouverte
Headers lourds à chaque échange Frames légères (2 à 14 octets d'overhead)
Protocole http:// / https:// Protocole ws:// / wss://
Stateless Stateful (connexion persistante)

Format des frames WebSocket

Les données sont transmises sous forme de frames, beaucoup plus légères que les messages HTTP :

  • FIN : indique si c'est la dernière frame du message
  • Opcode : type de frame (texte, binaire, ping, pong, close)
  • Mask : les données client→serveur sont masquées (sécurité)
  • Payload length : longueur des données (7, 16 ou 64 bits)
  • Payload data : les données proprement dites

Serveur WebSocket basique

import { WebSocketServer, WebSocket } from 'ws';
import { createServer } from 'http';
import express from 'express';

const app = express();
const server = createServer(app);
const wss = new WebSocketServer({ server });

wss.on('connection', (ws: WebSocket) => {
  console.log('Nouveau client connecté');

  ws.on('message', (data: Buffer) => {
    const message = data.toString();
    console.log('Message reçu :', message);

    // Renvoyer le message à tous les clients connectés
    wss.clients.forEach(client => {
      if (client.readyState === WebSocket.OPEN) {
        client.send(message);
      }
    });
  });

  ws.on('close', () => console.log('Client déconnecté'));
  ws.on('error', (err) => console.error('Erreur :', err));
});

server.listen(8080);

Client WebSocket natif (navigateur)

const socket = new WebSocket('ws://localhost:8080');

socket.addEventListener('open', () => {
  console.log('Connecté au serveur WebSocket');
  socket.send(JSON.stringify({ type: 'hello', text: 'Bonjour !' }));
});

socket.addEventListener('message', (event: MessageEvent) => {
  const data = JSON.parse(event.data);
  console.log('Message reçu :', data);
  displayMessage(data);
});

socket.addEventListener('close', (event: CloseEvent) => {
  console.log(`Connexion fermée (code: ${event.code})`);
});

socket.addEventListener('error', () => {
  console.error('Erreur WebSocket');
});

Chat bidirectionnel : exemple complet

interface ChatMessage {
  author: string;
  text: string;
  timestamp: number;
}

wss.on('connection', (ws: WebSocket) => {
  ws.on('message', (data: Buffer) => {
    try {
      const message: ChatMessage = JSON.parse(data.toString());

      // Diffuser à tous les autres clients
      wss.clients.forEach(client => {
        if (client !== ws && client.readyState === WebSocket.OPEN) {
          client.send(JSON.stringify(message));
        }
      });
    } catch {
      ws.send(JSON.stringify({ error: 'Format invalide' }));
    }
  });
});
